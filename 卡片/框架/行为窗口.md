## 定义

行为窗口（Behavior Window）是 PET 的基础分析工具。把孩子所有可能的行为想象为一个长方形窗口，用一条分割线分成上下两个区域：

```
┌──────────────────────────┐
│                          │
│    可接受的行为            │ ← 父母觉得 OK 的行为
│                          │
├─ ─ ─ ─ 分割线 ─ ─ ─ ─ ─ ┤ ← 这条线会上下浮动
│                          │
│    不可接受的行为          │ ← 父母觉得不 OK 的行为
│                          │
└──────────────────────────┘
```

## 构成要素

- **行为**：孩子实际做的事和说的话（客观事实），而非你对行为的评判
- **分割线**：并非固定不变，会受三个因素影响而上下浮动（见 [[接受感三因素模型]]）
- **可接受区**：父母内心真正感到 OK 的行为
- **不可接受区**：父母内心真正感到不 OK 的行为

## 运作逻辑

行为窗口的核心用途是帮助父母做出两个关键判断：

1. **这个行为对我来说是可接受的还是不可接受的？**（真实感受，不是"应该"怎么想）
2. **这个问题归属于谁？**（进而决定使用哪种技巧）

结合 [[问题归属]] 后的完整模型：

```
┌──────────────────────────┐
│  孩子拥有问题              │ → 用 [[积极倾听]]
│  (孩子不高兴/有困难)       │
├──────────────────────────┤
│  没有问题                  │ → 享受关系的时光
│  (双方都满足)              │
├──────────────────────────┤
│  父母拥有问题              │ → 用 [[面质性我-信息]]
│  (孩子的行为影响了父母)    │    或 [[第三法（没有输家的冲突解决）]]
└──────────────────────────┘
```

## 关键洞见

- 行为窗口的分割线位置因人而异、因时而异——这正是"[[父母是人不是神]]"的体现
- 任何时候父母都应问自己：**"这个问题归属于谁？"** 然后选择对应技巧
- P.E.T. 的目标就是缩小上下两个"有问题"区域，扩大中间"没有问题"的区域
