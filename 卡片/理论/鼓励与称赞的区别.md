## 核心论点

鼓励和称赞虽然都针对正面行为，但有本质区别。鼓励培养内在动力，称赞制造外在依赖。

## 对比

| | 鼓励 | 称赞 |
|---|---|---|
| 着重点 | 内在评价和努力过程 | 外在评价和最终结果 |
| 标准 | 个人的进步和努力 | 与他人比较、达到父母标准 |
| 时机 | 任何时候，包括失败时 | 只在表现优异时 |
| 效果 | 内在自我激励，接纳不完美的勇气 | 依赖外部认可，害怕失败 |
| 信念 | "你的努力有价值" | "你必须取悦我才是好的" |

## 称赞的负作用

- 有些孩子为了称赞而表现，价值感建立在取悦他人之上
- 得不到称赞的孩子更加沮丧，贬低自我能力
- 有些孩子识破称赞是控制工具，反而拒绝表现

## 鼓励用语示范

### 表示接纳
"我虽然不赞成你那样做，但我相信你有你的理由"

### 表示信心
"这也许很难，但我了解你的能力，我相信你能做得很好"

### 注重努力和进步
"这次也许分数上没有进步许多，但我知道你已经尽了力"

### 强调贡献和感激
"太谢谢了，你分担了许多家事"

## 容易让鼓励打折扣的话

- "你看，你只要用心就能做得好！"（暗示以前不用心）
- "你终于进步了，如果多努力些就更好了！"（画蛇添足）
- "你球打得不错，要是书也读得这样就好了。"（比较）
